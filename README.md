# TripShorts

## 프로젝트 소개
여행객이 여행지에서 촬영한 짧은 영상을 업로드하여 다른 사람들과 공유할 수 있는 플랫폼입니다. 사용자는 핸드폰 카메라로 찍은 영상을 업로드하고, 다른 이용자들의 영상을 틱톡처럼 스크롤하며 시청할 수 있습니다. 또한 좋아요와 댓글을 통해 소통할 수 있는 기능을 제공합니다.
